# Где реализованы 3D-объекты (молекулы и др.)

## Молекула воды

- **Файл:** `WaterMolecule.tsx`
- Там сделана модель H₂O: сферы (атомы) + цилиндры (связи), вращение через `useFrame`.

## Как добавить свои объекты

1. **Создай новый компонент** в этой папке (`src/components/`), например:
   - `Co2Molecule.tsx` — молекула CO₂  
   - `MethaneMolecule.tsx` — CH₄  
   - или любой другой объект (не только молекулы).

2. **В компоненте используй** примитивы Three.js через React Three Fiber:
   - `<mesh>`, `<sphereGeometry>`, `<boxGeometry>`, `<cylinderGeometry>`
   - `<meshStandardMaterial color="..." />`
   - Для связей можно переиспользовать идею компонента `Bond` из `WaterMolecule.tsx`.

3. **Подключи объект в сцену:** для страниц **элементов** используй `ElementScene.tsx` и XR там. Сцена уроков (`LessonScene.tsx`) сейчас только с видеоэкраном в VR/AR; дополнительные 3D-модели к урокам при необходимости подключай в отдельном Canvas или расширяй `LessonScene` вручную.

Все объекты внутри такой XR-сцены участвуют в VR и AR вместе с сессией.
