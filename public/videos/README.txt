Видео для режима «внутри VR» (текстура на 3D-экране)
=====================================================

1) Положи файл MP4 (или WebM) в эту папку:
   public/videos/

   Пример имени: lesson-water.mp4

2) В src/data/lessons.ts укажи ТОЛЬКО URL для браузера (начинается с /):
   immersiveVideoSrc: '/videos/lesson-water.mp4'

   НЕПРАВИЛЬНО: C:\Users\...\public\videos\file.mp4 — браузер не читает диск по букве диска.
   ПРАВИЛЬНО: файл лежит в public/videos/file.mp4 → в коде '/videos/file.mp4'

   После сборки (npm run build) файл окажется по адресу /videos/lesson-water.mp4 на сервере.

3) Почему не YouTube внутри VR?
   Ролик с YouTube нельзя легально и стабильно вывести как текстуру в WebGL.
   На странице урока используется встроенный плеер YouTube (2D).
   В шлеме рядом с 3D-сценой показывается отдельный файл MP4.

4) Требования к файлу:
   - Кодек H.264 в MP4 — лучшая совместимость с браузерами и Quest.
   - Разрешение 720p или 1080p достаточно.
   - Для автозапуска в VR видео в коде загружается с muted (без звука в браузере
     часто иначе autoplay блокируется). Звук можно включить отдельной настройкой позже.

5) Пока своего файла нет — в lessons.ts можно оставить демо-URL (HTTPS с CORS),
   см. комментарий в lessons.ts.
