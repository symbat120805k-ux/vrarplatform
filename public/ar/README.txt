AR (Mind AR) — кратко
=====================

Подробная инструкция по папкам и файлам для каждого из 10 элементов:

  public/media/ar/README.txt

Кратко: для элемента с номером N в `public/media/ar/N/` нужны card.png, targets.mind
(из компилятора Mind AR по этой же картинке) и video.mp4.

Компилятор: https://hiukim.github.io/mind-ar-js-doc/tools/compile/

Пути в коде: `src/data/scanConfig.ts`
